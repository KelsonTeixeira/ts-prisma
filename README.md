# ts-prisma
API using Typescript, Express, MongoDB and Prisma.

## endpoints
GET /api/users
CREATE /api/users
UPDATE /api/users/:id
DELETE /api/users/:id

## Run
```bash
  docker compose up -d
```

```bash
  npm i
```

```bash
  npx prisma generate
```

```bash
  npm run dev
```
